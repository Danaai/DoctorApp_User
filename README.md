# 🏥 Doctor Appointment Mobile App (User)

## 📱 Overview

DoctorApp User is a mobile application that allows patients to search for doctors, book appointments, and manage personal medical information efficiently. The app is designed to provide a smooth and user-friendly healthcare booking experience.

---

## 🚀 Features

* 🔐 User Authentication (Login/Register)
* 👤 Profile Management (Edit profile, change password)
* 📅 Book and manage appointments
* 🏥 View doctors and hospital information
* 📄 Medical history tracking
* ✅ Form validation (email, phone, password)

---

## 🛠 Tech Stack

* **Language:** Java
* **UI:** XML, ConstraintLayout
* **Architecture:** MVVM
* **Networking:** Retrofit (RESTful API)
* **Components:** RecyclerView, Navigation Component
* **Database:**  Firebase
